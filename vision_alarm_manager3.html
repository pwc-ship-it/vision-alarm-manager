⚡ Firebase 설정
Firebase Realtime Database를 연결하면 전 세계 어디서든 조치방안을 실시간으로 공유할 수 있습니다.
설정은 최초 1회만 필요합니다.
STEP 1 — Firebase 프로젝트 만들기
1. console.firebase.google.com 접속
2. 프로젝트 추가 클릭 → 이름 입력 (예: vision-alarm-manager) → 만들기
3. 구글 애널리틱스는 사용 안 함 선택해도 됩니다
STEP 2 — Realtime Database 생성
1. 왼쪽 메뉴 빌드 → Realtime Database 클릭
2. 데이터베이스 만들기 → 위치 선택 (아시아 권장: asia-southeast1) → 다음
3. 보안 규칙: 테스트 모드로 시작 선택 → 활성화
4. 생성된 DB URL 복사 (예: https://프로젝트명-default-rtdb.asia-southeast1.firebasedatabase.app)
STEP 3 — 보안 규칙 설정 (규칙 탭에 붙여넣기)
Realtime Database → 규칙 탭 → 아래 내용으로 교체 후 게시:
{ "rules": { "actions": { ".read": true, ".write": true }, "alarmEdits": { ".read": true, ".write": true }, "auditLog": { ".read": true, ".write": true }, "searchLog": { ".read": true, ".write": true } } }
STEP 4 — Database URL 입력
위에서 복사한 Database URL을 아래에 입력하세요:
※ 끝에 슬래시(/) 없이 입력하세요
💡 오프라인 모드: Firebase 미설정 또는 연결 실패 시 자동으로 로컬 저장 모드로 동작합니다. 나중에 연결하면 기존 로컬 데이터가 유지됩니다.
